I created this page to specifically check for the return of migrant species to England, such as the swift & cuckoo.

Enter a bird ID, one on more locations (by post-code or name), set an area radius, & then 'Fetch bird data'.

This simple site utilises the GraphQL API to access BirdWeather data from acoustic bird monitoring stations, mainly built around BirdNet-Pi & PUC systems.

The official BirdWeather website is: https://app.birdweather.com
