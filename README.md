# BirdWeather-Migration
Uses BirdWeather primarily to collect local data on specified migration birds.

Defo 'work in progress': I created this page to specifically check for the return of migrant species, such as the common swift, across a wide stretch of the southern coast of England.
But can be used for any species of interest, (I guess) anywhere in the world.

This simple html/Javascript page utilises the GraphQL API to access BirdWeather data from acoustic bird monitoring stations, mainly built around BirdNet-Pi & PUC systems.

The official BirdWeather website is: https://app.birdweather.com

In use:-
 - Enter a bird ID, one on more locations (by post-code or name), set an area radius, & then 'Fetch bird data'.
